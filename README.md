Bonsoir les taverniers!

Voici le script dont j'ai parlé à certains.

Il est fait pour être à la portée de tous au maximum, sans compétences particulières.

Il s'adresse à ceux qui ont une stack [radarr/(pas sonarr)/mediainfo/Qbittorrent]

Ce qu'il fait:

Parcourir la liste des films de radarr, intéroger lacale si même version, si oui, skip, si non, récupérer le nom orriginal de la release si modifié, générer .nfo avec médiainfo, générer torrent, placer en seed, publier sur la-cale.

En fonction du grad, vous devrez, ou le lancer le nombre de fois jusqu'à la limite, ou, l'automatiser en libre (à faire vous même).

C'est un peu plus de un mois de travail.

Je l'ai fait pour nous tous, pour enrichir les référencements et mettre en commun les trésors que nous avons.
