                   version avec sonarr pour v3.0
Pour aucune stack: seulement qbittorrent
        BLACK-FLAG version executable v0.9

Donc pour l’instant, seul la version formulaire web est publique.

Ce que fait le script:
Parcourir la liste des films de radarr, intéroger lacale si même version, si oui, skip, si non,
récupérer le nom original de la release si modifié, générer .nfo avec médiainfo, générer torrent, placer en seed, publier sur la-cale avec presentation.

Voici le script dont j'ai parlé à certains.

Il est fait pour être à la portée de tous au maximum, sans compétences particulières.

C'est un peu plus de un mois de travail.
Je l'ai fait pour nous tous, pour enrichir les référencements et mettre en commun les trésors que nous avons.
